# jokes_game

## English
This is a simple jokes game. The program presents a random joke when the user enters a number. Even if the same number is entered multiple times, a new random joke will be displayed each time.

## العربية
هذه لعبة نكت بسيطة. يقدم البرنامج نكتة عشوائية عندما يدخل المستخدم رقمًا. حتى إذا تم إدخال نفس الرقم عدة مرات، سيتم عرض نكتة عشوائية جديدة في كل مرة.
